# HerramientasIA

Web editorial y de descubrimiento para herramientasia.com.ar. El sitio está construido con Next.js App Router, TypeScript, Tailwind CSS, contenido local y SEO técnico listo para producción.

## Qué contiene

- Home editorial con categorías, comparativas, blog y FAQ.
- Directorio curado de herramientas IA.
- Páginas de categorías, comparativas, casos de uso, glosario y FAQ.
- Tres artículos completos del blog con fuentes y JSON-LD Article.
- Metadata por página, canonical, Open Graph, Twitter Cards y datos estructurados.
- `robots.txt`, `sitemap.xml`, `llms.txt`, `ai.txt` y manifest.
- Logo, favicon, apple touch icon y Open Graph image.

## Instalar

```bash
npm install
```

## Correr en local

```bash
npm run dev
```

El servidor de desarrollo usa el puerto `3007`.

## Construir

```bash
npm run build
```

## Producción

```bash
npm run start
```

Preview local: http://localhost:3007
